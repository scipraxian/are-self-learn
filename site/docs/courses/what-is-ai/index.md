# What Is AI?

**A self-paced online course for anyone who wants to understand how artificial intelligence actually works.**

You have heard the headlines. AI is writing code, generating images, passing bar exams, and threatening to take your job. But what is it actually doing? How does it read your words? How does it decide what to say back? Where does it physically live? And who is paying for all of this?

This course answers those questions without requiring you to write a single line of code. It is written for smart, curious people who happen to not be programmers. Every concept is explained from first principles, using analogies to things you already understand. And every module connects back to Are-Self — an open-source AI reasoning engine that you can run on your own hardware — as a concrete example of the ideas in action.

## Who This Is For

You. Specifically:

- You have used ChatGPT, Copilot, or similar tools and wondered what is happening behind the scenes
- You are a business leader making decisions about AI adoption
- You are a student exploring whether AI is a field you want to enter
- You are a parent trying to understand what your kids are using
- You are simply curious

You do not need to know how to program. You do not need a math background. You do need patience — some of these concepts are genuinely complex, and we take them seriously rather than dumbing them down.

## How This Course Works

There are six modules. Each module covers one fundamental concept. They build on each other — Module 1 introduces ideas that Module 3 depends on — so work through them in order the first time. After that, you can revisit any module independently.

Each module contains:

- **Explanations** — the core ideas, written in plain language with analogies
- **Are-Self connections** — how the concept shows up in a real AI system
- **Think about it** — questions to chew on (no graded assignments, just thinking prompts)
- **Going deeper** — optional pointers for readers who want more detail

There are no quizzes, no deadlines, no certificates. This is learning for its own sake.

## About Are-Self

Throughout this course, we use Are-Self as a running example. Are-Self is an open-source AI reasoning engine built with Python and Django. It is designed like a brain — literally. Its software components are named after brain regions: the Central Nervous System manages workflows, the Hypothalamus selects which AI model to use, the Hippocampus handles memory, and so on.

You do not need to install or run Are-Self to take this course. But knowing it exists — and that it is open-source, meaning anyone can inspect and modify it — gives you a concrete anchor for every abstract concept we discuss. When we talk about tokens, we can show you where Are-Self counts them. When we talk about models, we can show you how Are-Self chooses between them. When we talk about costs, we can show you how Are-Self tracks every penny.

Are-Self is built by Haunted Space Hotel and is free to use. You can find it at the Are-Self project page.

## Prerequisites

**None.** This course is designed for non-technical audiences. No programming, math, or prior AI knowledge is required. This is the universal entry point for anyone who wants to understand how artificial intelligence works.

## A Note on Honesty

AI is surrounded by hype. Companies overstate what their products can do. Headlines predict futures that may never arrive. This course is not marketing material. When something is impressive, we will say so. When something is limited, overhyped, or poorly understood, we will say that too. The goal is understanding, not enthusiasm.
