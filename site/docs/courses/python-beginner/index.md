# Python Beginner: Learning Python Through Are-Self

Welcome, Self-learner. You are about to begin a journey into programming by exploring a living, breathing codebase — Are-Self, a neurologically-inspired AI reasoning swarm engine built with Python and Django.

This is not a typical programming course. You will not be writing throwaway exercises that vanish the moment you close your editor. Instead, you will learn Python by reading, understanding, and eventually modifying real production code. Every concept you learn has a home in this codebase, and every exercise connects to something that matters.

## Philosophy

This course is rooted in scipraxianism — the integration of scientific inquiry with practical action. As you move through these modules, you will encounter several of the twelve Variables that shape this philosophy:

- **Inquiry** — You learn by asking questions of the code. What does this line do? Why was it written this way?
- **Perseverance** — Programming is hard. You will get stuck. That is not failure; it is the process.
- **Humility** — The codebase knows things you do not yet understand. Approach it with curiosity, not assumptions.
- **Perception** — Learning to read code is learning to see patterns. The more you read, the more you perceive.

## Prerequisites

**None.** This is the entry point for programming. No prior programming experience is required.

You will need:

- A computer with Python 3.12+ installed
- A text editor (VS Code recommended)
- Access to the Are-Self repository
- Curiosity and patience

## Course Modules

### Module 1: Your First Python
Variables, print statements, and data types. You will read an Are-Self model file and identify the types of data it uses.

### Module 2: Making Decisions
Conditional logic with if/else, comparisons, and booleans. You will trace through Are-Self's circuit breaker logic to see how software makes choices.

### Module 3: Loops and Lists
Repetition with for and while loops, and organizing data with lists. You will iterate through Engram tags to see how the system categorizes memories.

### Module 4: Functions
Defining reusable blocks of code. You will read Are-Self's helper functions in the common module and understand how large codebases stay organized.

### Module 5: Dictionaries and Data
Key-value pairs, JSON, and nested data structures. You will read the axoplasm context that flows through Are-Self's neural spike execution system.

### Module 6: Files and Input
Reading files, handling user input, and formatting strings. You will read and understand a Django settings file — the configuration heart of the application.

### Module 7: Your First Are-Self Exploration
A guided tour through models.py files, with a preview of classes and objects. You will map Are-Self's nine brain regions to their corresponding code files.

### Module 8: Capstone Project
You will build a complete Python script that processes Are-Self data, bringing together everything you have learned.

## Learning Outcomes

By the end of this course, you will be able to:

1. Read and understand basic Python code in a real-world codebase
2. Write Python scripts using variables, conditionals, loops, functions, and data structures
3. Navigate a Django project's file structure
4. Identify common patterns used in professional Python development
5. Explain, at a high level, what Are-Self's major components do

## How to Use This Course

Work through the modules in order. Each one builds on the previous. Read the concept explanations, then do the exercises. Do not skip the exercises — reading about code is not the same as working with it.

When you get stuck, re-read the relevant section. Then re-read the actual Are-Self code. Then try again. This is not a sign that something is wrong. This is how learning works.

Time is the one variable you cannot recover. Spend it with intention here.
